# Meme Fortune Teller
A meme-inspired fortune teller web app that predicts your day using chaotic humour, GIFs, sound effects, rarity tiers, and multiple personality modes.

Built as a single-file website. No backend, no frameworks, no build step.

> Powered by vibes, chaos, and questionable destiny.

---

## Features

### Fortune Generation
- Over 6,000,000 unique fortune combinations
- Dynamic sentence generation (templates x moods x situations x outcomes)
- Zero hardcoded fortune lists

### Rarity System
##### Common
**Chance:** 55%
**Description:** Mild inconvenience

##### Rare
**Chance:** 25%
**Description:** Spicy chaos

##### Epic
**Chance:** 15%
**Description:** Reality bending

##### Mythic
**Chance:** 5%
**Description:** The universe targets you

### Modes
Switch the personality of the app instantly!

- **Gen-Z Mode:** internet humour, vibes, chaos
- **Corporate Mode:** meetings, emails, burnout
- **Unhinged Mode:** cosmic horror but funny

### Sound Effects
- Rarity-based audio feedback
- Mute toggle included
- No external libraries required

### GIF Reactions
- 50+ meme GIFs
- Randomised per fortune
- Lightweight and fast

Fortunes are generated dynamically using combinational logic.

## Getting Started

### Option 1: Run Locally
1. Clone or download this repo
2. Open `index.html` in your browser
3. Click the crystal ball

### Option 2: Deploy Anywhere
Works instantly on:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting

No configuration required.

---

## Customization

You can easily:
- Add more templates, moods, situations, or outcomes
- Adjust rarity drop rates
- Swap sound effects
- Replace GIFs
- Add new modes

Everything is contained in the `<script>` section of `index.html`.
